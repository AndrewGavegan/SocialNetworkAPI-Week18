## Description
This application is a social network web application where users can share their thoughts, react to friendsâ€™ thoughts, and create a friend list. It uses Express.js for routing, a MongoDB database, and the Mongoose ODM. 

---
## Installation
To install the application yourself, fork the project, run `npm i` then run `npm start` in the command line.
